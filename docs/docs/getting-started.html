Getting Started

Installation

  1. Download the latest .msi from GitHub Releases.
  2. Run the installer.
  3. Kalam installs to Program Files and stores data in %APPDATA%/Kalam/.
  4. Kalam checks for updates automatically on startup.

First Launch — Onboarding Wizard

The onboarding wizard guides you through the app in 5 steps. It only appears once; you can access settings and profiles at any time from the sidebar.

  1. Welcome — Introduction to Kalam. Click "Get Started" or "Skip setup" to go straight to the dashboard.
  2. Tools You Can Manage — Shows the 6 supported tools: Rainmeter, YASB, GlazeWM, Zebar, Windhawk, Wallpaper. Click "Next" to continue.
  3. Need Help? — Links to Documentation and Discord. Click "Next".
  4. Your Workspace — Tour of the three main sections: Home (Dashboard), New Profile, Settings. Click "Next".
  5. Ready to Get Started — Choose "Create My First Profile" to go to the profile creator, or "Go to Dashboard" to see the main screen.

Navigating the App

  • Sidebar — Three navigation links: Home (Dashboard), New Profile, Settings
  • Custom titlebar — Minimize, maximize/restore, close buttons
  • Update banner — Appears at the bottom when a new version is available
