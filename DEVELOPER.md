# Kalam вҖ” Developer Documentation

## Overview

Kalam is a desktop **profile manager** for Windows customization tools. Users create profiles that bundle configurations for Rainmeter, YASB, wallpapers, GlazeWM, and Zebar. A single click applies everything at once.

**Tech stack:**
- **Desktop shell:** Tauri v2 (Rust backend + WebView frontend)
- **Frontend:** React 19 + Vite + React Router 7
- **System logic:** Python 3 sidecar (compiled to standalone .exe via PyInstaller)
- **Storage:** JSON files in `%APPDATA%/Kalam/`

## Data Schema

### `userSettings.json`
Stores global paths and the currently active profile.

### `userProfiles.json`
Stores the "target state" for each profile. For Windhawk, it will store a list of mods and their desired settings.
```json
{
  "Windhawk-Mods": [
    {
      "id": "taskbar-labels",
      "enabled": 1,
      "settings": { "show-labels": true }
    }
  ]
}
```

### `windhawkManifest.json` (Planned)
A system-specific file generated by the sidecar. It catalogs what mods are currently installed in the user's Windhawk instance.
```json
{
  "installedMods": [
    { "id": "taskbar-labels", "name": "Taskbar Labels" },
    { "id": "dark-mode-fix", "name": "Dark Mode Fix" }
  ]
}
```
